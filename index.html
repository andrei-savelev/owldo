Простой список задач "OwlDo"

Вся логика работы этого приложения реализована с помощью трех технологий, сонованных на JavaScript.

  • AngularJS
  • LocalStorage (HTML5)
  • JQuery

Практически весь код подробно задокументирован, поэтму понять принцип работы этого приложения не составит труда. Благодаря современным технологиям хранинея информации на клиенте, даже после закрытия окна или его перезагрузки, все ваши задачи сохраняться, и вы сможете вернуться к их исполнению в любое время.

В основе приложения - AngularJS, он отвечает за всю основную работу "OwlDo".

Хранение информации осуществляется на клиенте, в LocalStorage

Данное окно - результат работы нехитрого кода на JQuery
